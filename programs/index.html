Programs

Most of the software I use is free and open-source. All of them are privacy friendly (except windows). I like to follow a minimalistic lifestyle, so my choices of software will follow a “do one thing, but do it good” attitude. For more software recommendations check out My megathread.

Basic Software

Operating System

  • Artix Linux - This is my primary Linux distro and OS. It is a fork of Arch Linux without systemd.

  • Windows 10 LTSC - Unfortunately I’m still forced to use Windows for gaming so I have to use this proprietary garbage which is slightly better than standard windows 10 and windows 11 (pls don’t use windows 11 lol).

Terminal

  • st - Simple Terminal by suckless.org

Shell

  • Zsh - A shell that is better than bash.

Web browser

  • Librewolf - A fork of Firefox, focused on privacy, security and freedom.

  • Firefox - Probably the best browser. (I use it along with arkenfox’s user.js).

  • Brave - Chromium fork with better privacy settings. Don’t really recommend it but I use it when sites break/don’t load in Firefox.

Utilities

File manager

  • lf - A Terminal file manager.

Media player

  • MPV - The best media player.

Torrent client

  • qBittorrent - Probably the best torrent client.

  • Transmission - More minimalistic torrent client.

PDF viewer

  • Zathura - Minimal PDF viewer.

  • Sumatra - PDF viewer for Windows.

Image Viewer

  • sxiv - Simple X Image Viewer

  • ImageGlass - Image viewer for Windows.

Office Suite

  • Libreoffice - The best open-source office suite.

RSS Reader

  • Newsboat - A terminal RSS feed reader.
