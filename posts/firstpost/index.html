Welcome to my site!

It took me a while to learn Hugo and CSS but I’ve finally deployed my own site from scratch. This site was created completely from scratch using just Hugo, Html and CSS (No JS!). This is still a work in progress and I’ll keep updating this site as time goes on, so watch out for updates!

This site is based on minimalism and a ‘keep it simple’ mindset. I don’t plan on making this website fancy (bloated) with dumb JS, unnecessary animations, etc.

Check out the source code of this website on Github. If you have any suggestions, ideas or find any improvements, feel free to create an issue on Github or email me.
